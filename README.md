HDSD:

Cài python

Bước 1: Mở Command Prompt (CMD)
Nhấn phím Windows trên bàn phím (hoặc ô Search).

Gõ chữ cmd.

Thấy biểu tượng màu đen hiện lên, hãy chọn Run as Administrator (Chạy với quyền quản trị) để tránh lỗi phân quyền.

Bước 2: Nhập lệnh cài đặt
Trong cửa sổ CMD màu đen, bạn copy và dán dòng này vào rồi nhấn Enter:

Bash
pip install webdriver-manager selenium
Nếu nó hiện: Successfully installed... là bạn đã thành công!

Nếu nó báo lỗi: 'pip' is not recognized..., nghĩa là lúc cài Python bạn chưa tích vào ô "Add Python to PATH". Đừng lo, hãy thử gõ lệnh này thay thế:
python -m pip install webdriver-manager selenium
